# I&I

I&I is a social media application.

## Features

- Admin Login
- User Registration
- User Login
- Text Posts
- Photo Upload
- Video Upload
- Audio Upload
- 30 Days Free Trial
- $2 Monthly Subscription

## Technologies

- Node.js
- Express.js
- JSON Database
- Termux
- Acode
